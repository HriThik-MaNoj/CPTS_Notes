> Removes the need to configure Proxychains

- **Only works for pivoting over SSH**

#### Installing sshuttle
```python
sudo apt-get install sshuttle
```

#### Running sshuttle
```python
HrithikManoj000@htb[/htb]$ sudo sshuttle -r ubuntu@10.129.202.64 172.16.5.0/23 -v
```

![[Pasted image 20260319151027.png]]

With this command, sshuttle creates an entry in our `iptables` to redirect all traffic to the `172.16.5.0/23` network through the pivot host.

**We can now use any tool directly without using proxychains.**
