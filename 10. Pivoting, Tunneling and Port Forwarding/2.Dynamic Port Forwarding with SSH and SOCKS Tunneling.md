**Port Forwarding**  : Technique that allows us to redirect a communication request from one port to another.

## SSH Local Port Forwarding
![[Pasted image 20260317075625.png]]
- Attack host : 10.10.15.5
- Target : 10.129.15.50
- when scanning the target, we got the following result:
![[Pasted image 20260317075847.png]]
- We can see that ssh is open.
- In order to access the mysql server, there are two options, `ssh` into the target machine and access it from there or `port forward` it to our local port say 1234 and access it locally.

#### Executing the Local Port Forward
```bash
ssh -L 1234:localhost:3306 ubuntu@10.129.202.64
```
- **-L** : Tells the SSH client to request the SSH serer to forward all the data we send via the port 1234 to `localhost:3306`.
#### Confirming Port Forward with Nmap
```bash
nmap -v -sV -p1234 localhost
```
