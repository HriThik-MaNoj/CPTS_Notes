- From a foothold on a domain-joined Windows host, the [DomainPasswordSpray](https://github.com/dafthack/DomainPasswordSpray) tool is highly effective

### When to Use (From a Windows Host)

- Have a **foothold on a domain-joined Windows host**
- Want to spray from a **managed Windows device** (on-site or VM)
- Gained **initial foothold** and want to escalate privileges via credential spraying

---

### Key Advantage (Authenticated to Domain)

When already authenticated, the tool **automatically**:

- Generates a **user list from Active Directory**
- Queries the **domain password policy**
- **Excludes accounts** within one attempt of lockout ✅

---

### Common Flags

|Flag|Purpose|
|---|---|
|`-UserList`|Supply a manual user list (skip if domain-joined)|
|`-Password`|Single password to spray|
|`-OutFile`|Save results to a file for later use|

> If **domain-joined** → skip `-UserList`, let the tool auto-generate from AD
