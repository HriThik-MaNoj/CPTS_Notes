#### Username enumeration using creds
 - since we now got creds, lets try to get the user listing using the creds obtained.
- Then we can do password spraying on them.
```python
sudo crackmapexec smb 172.16.7.3 --users -u AB920 -p weasal > users.txt
cat users.txt | awk '{ print $5 }' | cut -d '\' -f2 > valid_users.txt
```
