## Enumeration
#### Host - MX Records
```
host -t MX hackthebox.eu
```

```
host -t MX microsoft.com
```


#### DIG - MX Records
```
dig mx plaintext.do | grep "MX" | grep -v ";"
```

```
dig mx inlanefreight.com | grep "MX" | grep -v ";"
```

#### Host - A Records
```
host -t A mail1.inlanefreight.htb.
```

## Misconfigurations

#### Authentication
- `SMTP` server has different commands that we can use to enumerate usernames `VRFY`, `EXPN` and `RCPT TO`.
- `VRFY` this command instructs the receiving SMTP server to check the validity of a particular email username.

![[Pasted image 20260314152805.png]]

- `EXPN` is similar to `VRFY`, except that when used with a distribution list, it will list all users on that list.
![[Pasted image 20260314152934.png]]

- `RCPT TO` identifies the recipient of the email message
