- Samba is a Unix/Linux-based open-source implementation of the SMB protocol.


### Enumeration

```
sudo nmap 10.129.14.128 -sV -sC -p139,445
```

## Misconfigurations

#### Anonymous Authentication

- Most tools that interact with SMB allow null session connectivity, including `smbclient`, `smbmap`, `rpcclient`, or `enum4linux`

##### SMB client null session attack

```
smbclient -N -L //10.129.14.128
```

##### Smbmap null session attack

```
smbmap -H 10.129.14.128
```
