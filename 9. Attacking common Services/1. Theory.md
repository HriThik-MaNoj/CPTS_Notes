## SMB
- we can interact with SMB in windows in multiple ways.
- [winkey] + [R] to open dialogue box and type the share location eg: `\\192.168.220.129\Finance\`
- If anyonymous login in enabled, we wont get password prompt, else we will..
![[Pasted image 20260307104446.png|526]]

#### Windows CMD - DIR

```
dir \\192.168.220.129\Finance\
```

#### Windows CMD - Net Use

```
net use n: \\192.168.220.129\Finance
```

```
net use n: \\192.168.220.129\Finance /user:plaintext Password123
```

#### Command to determine how many files exist in that share

```
dir n: /a-d /s /b | find /c ":\"
```

#### Windows CMD - Findstr

```
findstr /s /i cred n:\*.*
```
