## Verify RFI
-  Any remote URL inclusion in PHP would require the `allow_url_include` setting to be enabled.
- we can check this similar to how we did for LFI refer to the LFI note ([[1. Local File Inclusion]])for additional details
```python
echo 'W1BIUF0KCjs7Ozs7Ozs7O...SNIP...4KO2ZmaS5wcmVsb2FkPQo=' | base64 -d | grep allow_url_include allow_url_include = On
```
