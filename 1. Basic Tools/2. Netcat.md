# Netcat (nc)

Network utility for interacting with TCP/UDP ports. Primary use: quick shell connectivity and banner grabbing.

## Connect to a listening host
```bash
nc -nv 10.10.10.10 22
# Example output: SSH-2.0-OpenSSH_8.4p1 Debian-3
```

## Listen for reverse shells
```bash
nc -nvlp 1234
```
- -n: no DNS
- -v: verbose
- -l: listen
- -p: port
