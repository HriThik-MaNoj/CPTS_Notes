# Tmux

Terminal multiplexer for managing multiple sessions, windows, and panes.

Start tmux:
```bash
tmux
```

Prefix to send commands: Ctrl+b
- c: new window
- t: show time
- [window index]: jump to window
- %: vertical split
- ": horizontal split
- Arrow keys after prefix: switch panes
