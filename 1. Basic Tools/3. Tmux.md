
- Just type tmux to start using it
- key combo to input commands : `[CTRL + B]`
	- press c for new window
	- press **t** to show the time
	- enter the session number to move to that session
	- Vertical split : `[SHIFT + %]`
	- Horizontal split : `[SHIFT + "]`
	- Switch between panes by hitting the prefix and then the `left` or `right` arrows for horizontal switching or the `up` or `down` arrows
