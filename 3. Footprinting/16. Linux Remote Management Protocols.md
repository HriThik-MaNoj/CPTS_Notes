## Rsync
- Fast and efficient tool for locally and remote copying files.
- portÂ `873`

**Footprinting**
```bash
sudo nmap -sV -p 873 127.0.0.1
```

**Probing for Accessible shares**

```bash
nc -nv  127.0.0.1 873
```

**Enumerating an Open share**

```bash
rsync -av --list-only rsync://127.0.0.1/dev
```

**Sync all files to our attack host with this command :** 

```bash
rsync -av rsync://127.0.0.1/dev
#If rsync is configured to use SSH to transfer files, we would modify our command to include -e ssh flag or -e ssh -p2222
```
