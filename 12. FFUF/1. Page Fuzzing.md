## Extension Fuzzing
```python
ffuf -w /opt/useful/seclists/Discovery/Web-Content$/web-extensions.txt:FUZZ -u http://SERVER_IP:PORT/blog/indexFUZZ
```
## Page Fuzzing
```python
ffuf -u http://154.57.164.83:31252/blog/FUZZ -w /usr/share/seclists/Discovery/Web-Content/DirBuster-2007_directory-list-lowercase-2.3-big.txt \
-e .php,.html,.txt \
-mc 200
```
## Recursive Fuzzing
```python
ffuf -w /opt/useful/seclists/Discovery/Web-Content/directory-list-2.3-small.txt:FUZZ -u http://154.57.164.83:31252/FUZZ -recursion -recursion-depth 1 -e .php -v
```

## Vhost Fuzzing

## Filtering 
- Filtering out certain length responses
```python
ffuf -w /opt/useful/seclists/Discovery/DNS/subdomains-top1million-5000.txt:FUZZ -u http://154.57.164.78:31110/ -H 'Host: FUZZ.academy.htb' -fs 900

# Filters out response length 900
```
