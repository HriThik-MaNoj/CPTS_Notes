## Extension Fuzzing
```python
ffuf -w /opt/useful/seclists/Discovery/Web-Content$/web-extensions.txt:FUZZ -u http://SERVER_IP:PORT/blog/indexFUZZ
```
## Page Fuzzing
```python
ffuf -u http://154.57.164.83:31252/blog/FUZZ -w /usr/share/seclists/Discovery/Web-Content/DirBuster-2007_directory-list-lowercase-2.3-big.txt \

```

## Filtering 
- Filtering out certain length responses
```python
ffuf -w /opt/useful/seclists/Discovery/DNS/subdomains-top1million-5000.txt:FUZZ -u http://154.57.164.78:31110/ -H 'Host: FUZZ.academy.htb' -fs 900

# Filters out response length 900
```
- Filtering by content
```python
ffuf -w /opt/useful/SecLists/Discovery/Web-Content/directory-list-2.3-small.txt:FUZZ -u http://faculty.academy.htb:STMPO/FUZZ -recursion -recursion-depth 1 -e .php,.phps,.php7 -fs 287 -mr "You don't have access!" -t 100

#Filter out the response that has that particular message
```

## Parameter Fuzzing - GET
```python
ffuf -w /opt/useful/seclists/Discovery/Web-Content/burp-parameter-names.txt:FUZZ -u http://154.57.164.78:31110/admin/admin.php?FUZZ=key -fs xxx
```

## Parameter Fuzzing - POST
- Different from GET parameter fuzzing because POST requests are not passed with the URL and cannot be simply appended after a ? symbol.
- POST requests are passed in the data field within the HTTP request.

```python
ffuf -w /opt/useful/seclists/Discovery/Web-Content/burp-parameter-names.txt:FUZZ -u http://admin.academy.htb:PORT/admin/admin.php -X POST -d 'FUZZ=key' -H 'Content-Type: application/x-www-form-urlencoded' -fs xxx
```
## Value Fuzzing
```python
# Creating a wordlist for id's
for i in $(seq 1 1000); do echo $i >> ids.txt; done
```

```python
ffuf -w ids.txt:FUZZ -u http://154.57.164.69:31167/admin/admin.php -X POST -d 'id=FUZZ' -H 'Content-Type: application/x-www-form-urlencoded' -fs xxx
```
