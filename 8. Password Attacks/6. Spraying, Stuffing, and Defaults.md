## SSH Local port forwarding

#### MySQL Remote Access via SSH Tunnel - Quick Reference

###### Scenario

- **Target:** MySQL server bound to localhost only (`bind-address = 127.0.0.1`)
- **Access:** SSH credentials for target machine
- **Goal:** Access MySQL remotely and brute-force credentials

## Step 1: Create SSH Tunnel (Attack Host)

bash


## Step 2: Access MySQL Through Tunnel (New Terminal on Attack Host)

bash

```bash
# Manual connection attempts
mysql -h 127.0.0.1 -u root -p
mysql -h 127.0.0.1 -u sam -p
mysql -h 127.0.0.1 -u admin -p
```
