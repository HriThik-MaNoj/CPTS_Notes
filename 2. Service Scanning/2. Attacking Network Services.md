# Attacking Network Services

## Banner Grabbing
Useful for quick fingerprinting.

### Using Nmap
```bash
nmap -sV --script=banner <target>
# example
nmap -sV --script=banner -p21 10.10.10.0/24
```

### Using Netcat
```bash
nc -nv <target> <port>
```

## FTP
File Transfer Protocol. Supports common commands (cd, ls, get).

### Connect
```bash
ftp -p <target>
```

### Download a file
```bash
get <filename>
```

## SMB
Server Message Block. Common on Windows; useful for lateral movement and can leak sensitive data.

### Nmap scripts
```bash
nmap --script smb-os-discovery.nse -p445 10.10.10.40
```

### Enumerate shares with smbclient
```bash
smbclient -N -L \\10.129.42.253
# -L: list shares
# -N: no password prompt
```

### Connect as guest
```bash
smbclient \\10.129.42.253\users
```

### Connect with credentials
```bash
smbclient -U bob \\10.129.42.253\users
# enter password when prompted
```

## SNMP
Simple Network Management Protocol. v1/v2c lack encryption/authentication (v3 adds them). Can reveal processes, versions, routing info.

### snmpwalk
```bash
snmpwalk -v 2c -c public 10.129.42.253 1.3.6.1.2.1.1.5.0
snmpwalk -v 2c -c private 10.129.42.253
```

### onesixtyone (brute-force community strings)
```bash
onesixtyone -c dict.txt 10.129.42.254
```
