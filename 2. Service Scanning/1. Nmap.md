# Nmap

Default scan type is TCP unless UDP is requested.

## Useful flags
```text
-sC   Run default scripts (NSE)
-sV   Detect service versions
-sU   UDP scan
-p-   Scan all ports
-O    OS detection
-A    Aggressive scan (OS, versions, scripts, traceroute)
```

## Examples
```bash
nmap -sC -sV -oN nmap_initial.txt 10.10.10.10
nmap -p- -sV --min-rate 2000 -oN nmap_allports.txt 10.10.10.10
```
