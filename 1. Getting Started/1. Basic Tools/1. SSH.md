# SSH

Secure Shell for remote access, port forwarding/tunneling, and file transfer.

- More stable than a raw reverse shell
- Default port: 22

## Quick login
```bash
ssh bob@10.10.10.10
# enter password when prompted
```

## Common options
- -i identity_file: use a private key
- -p port: specify port
- -L local_port:remote_host:remote_port: local port forward
- -N: do not execute remote commands (useful with forwarding)
