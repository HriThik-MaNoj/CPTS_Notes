# Web Enumeration

## Directory/File enumeration
### Gobuster
GoBuster performs DNS, vhost, and directory brute-forcing; can also enumerate AWS S3 buckets.
```bash
gobuster dir -u http://10.10.10.121/ -w /usr/share/seclists/Discovery/Web-Content/common.txt
```

## Banner grabbing / Web server headers
Similar to [[2. Attacking Network Services]]. Use curl to retrieve server headers.
```bash
curl -IL https://www.inlanefreight.com
```

## WhatWeb
Extracts versions of web servers, frameworks, and applications.
```bash
whatweb 10.10.10.121
```

## Certificates
SSL/TLS certificates can reveal details (emails, orgs) useful for OSINT/phishing when in scope.

## robots.txt
Instructs crawlers about access; may reveal private files/admin pages.

## Source code
Inspect source for credential leaks or links to hidden paths.
